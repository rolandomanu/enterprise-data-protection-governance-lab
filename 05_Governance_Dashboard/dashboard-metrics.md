# Governance Dashboard Metrics

## Key Performance Indicators (KPIs)

### Data Protection

- DLP Incidents per Month
- Sensitive Data Detections
- Policy Violations
- False Positive Rate

### Insider Risk

- Insider Risk Cases Opened
- Insider Risk Cases Closed
- High-Risk User Activities

### Governance

- Risks Identified
- Risks Mitigated
- Risks Accepted

### Compliance

- Controls Implemented
- Controls Reviewed
- Audit Findings

### Vulnerability Management

- Critical Vulnerabilities
- Remediation Rate
- Average Time to Remediate
