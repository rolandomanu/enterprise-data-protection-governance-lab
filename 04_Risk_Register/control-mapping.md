# Security Control Mapping

## Purpose

Map enterprise security risks to controls aligned with recognized frameworks.

| Risk ID | Control | Framework |
|----------|----------|-----------|
| R-001 | Data Loss Prevention (DLP) | NIST PR.DS / ISO 27001 A.5 |
| R-002 | Insider Risk Monitoring | NIST DE.CM |
| R-003 | RBAC & Access Reviews | NIST PR.AC |
| R-004 | Vulnerability Management | NIST ID.RA |
| R-005 | Endpoint Security Controls | NIST PR.PT |
| R-006 | Data Classification Policy | ISO 27001 Information Protection |
| R-007 | Audit Evidence Management | ISO 27001 Internal Audit |
| R-008 | Information Classification | ISO 27001 Asset Management |

## Framework References

- NIST Cybersecurity Framework (CSF)
- ISO/IEC 27001
