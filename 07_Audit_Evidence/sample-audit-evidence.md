# Sample Audit Evidence

## Purpose

Provide examples of evidence that may be collected during security audits and compliance assessments.

## Evidence Examples

### Access Review Evidence

Description:
Quarterly review of privileged accounts.

Evidence:
- User access reports
- Approval records
- Access review documentation

Control Objective:
Ensure least privilege access.

---

### DLP Monitoring Evidence

Description:
Review of DLP alerts and incidents.

Evidence:
- DLP incident reports
- Alert investigation records
- Policy violation documentation

Control Objective:
Protect sensitive information.

---

### Vulnerability Management Evidence

Description:
Validation of remediation activities.

Evidence:
- Vulnerability scan reports
- Patch deployment records
- Risk acceptance approvals

Control Objective:
Reduce security exposure.

---

### Security Awareness Evidence

Description:
Completion of annual security training.

Evidence:
- Training records
- Completion certificates
- Attendance logs

Control Objective:
Improve security awareness.
