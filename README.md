# Enterprise Data Protection & Governance Lab

Enterprise-style cybersecurity governance project focused on Data Protection, Data Loss Prevention (DLP), Insider Risk Management, and Security Governance aligned with Microsoft Security concepts, ISO/IEC 27001, and NIST Cybersecurity Framework (CSF).

## Project Objectives

- Simulate enterprise data protection programs
- Develop DLP governance controls
- Build insider risk management scenarios
- Create governance dashboards and risk registers
- Align security controls with ISO 27001 and NIST CSF
- Strengthen data protection and compliance visibility

## Areas Covered

- Data Classification
- Data Loss Prevention (DLP)
- Insider Risk Management
- Governance & Compliance
- Risk Management
- Audit Readiness
- Microsoft Security / Purview Concepts
- Security Policy Development

## Technologies & Concepts

- Microsoft Purview
- DLP Policies
- Regex & Boolean Logic
- NIST CSF
- ISO/IEC 27001
- IAM / RBAC
- Governance Frameworks
- Risk Registers
- Security Operations
