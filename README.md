# Enterprise Data Protection & Governance Lab

Enterprise-style cybersecurity governance project focused on:

- Data Protection
- Data Loss Prevention (DLP)
- Insider Risk Management
- Security Governance
- Risk & Compliance
- Microsoft Security / Purview concepts

## Objectives

- Simulate enterprise DLP governance programs
- Develop data classification policies
- Create insider risk scenarios
- Build governance dashboards and risk registers
- Align controls with ISO 27001 and NIST CSF

## Technologies & Concepts

- Microsoft Purview
- DLP
- Information Protection
- Insider Risk
- Regex & Boolean Logic
- NIST CSF
- ISO 27001
- IAM
- Governance & Compliance
