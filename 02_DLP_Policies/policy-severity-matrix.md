# DLP Severity Matrix

| Severity | Description | Example |
|-----------|-------------|----------|
| Critical | Regulatory or highly sensitive data exposure | SSN, Payment Cards |
| High | Confidential information exposure | Customer Records |
| Medium | Internal information sharing violation | Internal Reports |
| Low | Minor policy violation | Public Information Misuse |

## Response Guidelines

### Critical
- Immediate investigation
- Incident response activation
- Management notification

### High
- Security review
- User validation
- Case documentation

### Medium
- Monitoring
- Awareness training

### Low
- Documentation only
