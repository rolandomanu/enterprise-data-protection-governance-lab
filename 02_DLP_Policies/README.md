# Data Loss Prevention (DLP) Policies

This section contains sample DLP policies designed to protect sensitive information and reduce the risk of unauthorized disclosure, data leakage, and insider threats.

## Objectives

- Detect sensitive data
- Reduce data exfiltration risks
- Support compliance requirements
- Improve data visibility and governance

## Covered Areas

- Social Security Numbers (SSN)
- Credit Card Information
- Source Code Protection
- Sensitive Keywords
- Data Handling Violations
