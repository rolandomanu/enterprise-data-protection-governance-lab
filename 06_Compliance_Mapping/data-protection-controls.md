# Data Protection Controls

## Objective

Protect sensitive information from unauthorized access, disclosure, modification, or loss.

## Control Examples

| Control | Purpose |
|----------|----------|
| Data Classification | Identify sensitive information |
| DLP Policies | Prevent data leakage |
| Encryption | Protect data confidentiality |
| Access Reviews | Enforce least privilege |
| Audit Logging | Support investigations |
| Insider Risk Monitoring | Detect risky user behavior |

- Governance
