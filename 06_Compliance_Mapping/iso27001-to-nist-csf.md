# ISO 27001 to NIST CSF Mapping

## Purpose

This document demonstrates a simplified mapping between ISO/IEC 27001 controls and NIST Cybersecurity Framework functions.

| ISO 27001 Control Area | NIST CSF Function |
|------------------------|------------------|
| Information Security Policies | Govern (GV) |
| Access Control | Protect (PR) |
| Asset Management | Identify (ID) |
| Incident Management | Respond (RS) |
| Monitoring Activities | Detect (DE) |
| Business Continuity | Recover (RC) |

## Benefits

- Supports audit preparation
- Simplifies framework alignment
- Improves governance consistency
