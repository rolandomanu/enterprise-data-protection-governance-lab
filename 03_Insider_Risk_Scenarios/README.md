# Insider Risk Scenarios

This section contains simulated insider risk scenarios designed to demonstrate how organizations can identify, investigate, and respond to potential data protection and data exfiltration risks.

## Objectives

- Identify risky user behaviors
- Support data protection initiatives
- Improve governance visibility
- Reduce data exfiltration risks
- Align with Microsoft Purview Insider Risk Management concepts

## Scenarios

- USB Data Exfiltration
- Personal Cloud Uploads
- Mass Download Before Resignation
- Email Forwarding of Confidential Information

## References

- Microsoft Purview Insider Risk Management
- Data Loss Prevention (DLP)
- NIST Cybersecurity Framework (CSF)
