# Insider Risk Scenario: USB Data Exfiltration

## Scenario Overview

An employee copies a large volume of confidential files to a USB storage device outside normal business activities.

## Risk Indicators

- Unusual file copying activity
- Large number of files transferred
- Sensitive data involved
- Activity outside business hours

## Potential Impact

- Loss of confidential information
- Regulatory compliance issues
- Reputational damage

## Recommended Actions

- Generate security alert
- Investigate user activity
- Review file access logs
- Validate business justification

## Severity

High

## Related Controls

- DLP Monitoring
- Endpoint Monitoring
- Access Control
- Insider Risk Management
