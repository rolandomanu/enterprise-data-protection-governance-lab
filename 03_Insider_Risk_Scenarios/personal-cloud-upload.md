# Insider Risk Scenario: Personal Cloud Upload

## Scenario Overview

An employee uploads confidential information to a personal cloud storage service.

## Risk Indicators

- Upload to unauthorized cloud provider
- Sensitive documents transferred
- Repeated transfer activity

## Potential Impact

- Unauthorized disclosure
- Data leakage
- Loss of organizational control

## Recommended Actions

- Alert Security Team
- Investigate uploaded content
- Validate business need
- Block future uploads if necessary

## Severity

High

## Related Controls

- DLP Policies
- Cloud Security Controls
- Data Classification
